Usability:

  1. Image content is the largest factor this site has, so reducing the image size to make sure that they are appropriately sized is a good way to do this. It is also recommended to serve the images in a next-gen format.
  2. Removing unused javascript code will help increase the speed by reducing bytes consumed by the netwrok activity.

Accessibility:

  1. There are multiple empty alt text that need to be filled in. Adding these will give a description of the img if it fails to load.
  2. There is a low background contrast under Chicago Zoological Society that is hard to read. Changing the text color or giving a shadow may help.
